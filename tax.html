Housing Taxation System

The Housing Taxation System is a web application designed to calculate and manage housing taxes for homeowners and local governments. Homeowners can register on the platform to pay their taxes, while government officials, acting as admins, can approve registrations and apply penalties for delayed payments.

The technologies used include;

  • HTML, CSS, JavaScript
  • PHP
  • MySQL
  • PayPal API

Features

  • Tax calculation and management
  • Tax payment integration
  • User and admin dashboards
