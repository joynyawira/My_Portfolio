Refund and Compensation Form

The Refund and Compensation Form is a web application designed to manage refund and compensation requests for customers. This platform allows users to submit refund and compensation claims easily through a user-friendly interface. I developed and hosted this web form, ensuring it is secure and efficient for both users and administrators.

Technologies Used

  • HTML, CSS
  • PHP
  • MySQL
  • PHP mailer

Features

  • Refund request submission
  • Refund processing and tracking
  • Admin management interface
